Question 3:
The options that I used for the API call were fetch and get. Both options
ultimately do the same thing, but go about it in different ways. Using "get",
creates a request object from the url, opens the request, creates an onreadystatechange
event to the callback, and then sends the request. Fetch on the other hand, does many of
same operations, only it streamlines the request by consolidating many of the common
elements of an XMLHttpRequest.
